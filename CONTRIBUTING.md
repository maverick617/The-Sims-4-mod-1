# Contributing

Thanks for helping improve this Sims 4 mod project.

## Before You Contribute

Please make sure your contribution follows these rules:

- Your textures, meshes, icons, names, and descriptions must be original or properly licensed.
- Do not submit paid custom content from other creators.
- Do not submit brand logos, celebrity photos, copyrighted characters, or copied artwork.
- Test your change in game when possible.

## Suggested Contribution Types

- New swatches
- Texture improvements
- Mesh fixes
- Translation updates
- Bug reports
- Documentation improvements

## Pull Request Checklist

- [ ] I tested the mod or custom content in The Sims 4.
- [ ] I added or updated documentation if needed.
- [ ] I only included assets I have the right to share.
- [ ] I included screenshots for visual changes.
- [ ] I updated release notes if this is user-facing.
