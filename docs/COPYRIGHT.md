# Copyright Notes

This project is intended to be safe for public GitHub release.

## Good Sources

Use assets from:

- Artwork you created yourself
- Photos you took yourself
- Public domain sources
- Open licensed sources that allow remixing and redistribution
- Contributors who explicitly agree to the project license

## Avoid

Do not include:

- Other creators' paid custom content
- Reuploaded `.package` files from other creators
- Brand logos or trademarks used as decoration
- Celebrity images
- Copyrighted characters from films, games, comics, or anime
- AI-generated images that imitate a living artist or protected character

## Recommended Disclaimer

Use this disclaimer in the README, release pages, and download pages:

```text
This is an unofficial fan-made project. It is not endorsed by or affiliated with Electronic Arts, Maxis, or The Sims.
```

## Licensing Split

Recommended default:

- Code, scripts, XML tuning, and documentation: MIT License
- Original visual assets: CC BY 4.0
- Final `.package` release files: same permissions as the source assets inside them, unless stated otherwise

If a package contains assets from multiple sources, document every source in `docs/ASSET_CREDITS.md`.
